## JOG Feed

Modo de operación en el cual la posición del robot es ajustada manualmente por el operario. Existen 6 diferentes modos de operación en con JOG:

- JOINT
- TOOL
- XYZ
- 3-XYZ
- CYLNDER
- WORK
